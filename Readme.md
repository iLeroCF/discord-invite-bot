# Discord Invite Tracker Bot

Bu proje, Discord sunucularında **hangi kullanıcının hangi davet linkiyle katıldığını** tespit eden,  
**event tabanlı**, **temiz dosya yapısına sahip** bir Discord botudur.

Bot, davet bilgilerini otomatik takip eder ve yöneticiler tarafından belirlenen bir kanala mesaj atar.

---

## Özellikler

- Event tabanlı mimari
- Komutlar `commands/` klasöründe
- Eventler `events/` klasöründe
- Temiz ve sade `index.js`
- Davet linkiyle katılan kullanıcıyı tespit etme
- Daveti oluşturan kişiyi ve kullanım sayısını gösterme
- Sadece admin yetkisi olanların ayar yapabilmesi
- Slash command desteği
